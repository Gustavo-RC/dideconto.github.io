Neste momento já temos o nosso Array de Créditos finalizado, agora vamos vincular ele ao Ciclo de Pagamentos.

Em nosso create do Ciclo de Pagamentos já possui o nosso método de criação que chama nosso serviço salva o nosso back-end, para dar continuidade vamos deletar o comentário criado e fazer com que nosso cycle receba nosso atributo credits a nossa variável credits (Array).

Próximo passo é incluir a variável da data, para isso precisamos realizar alguns ajustes e inclusões.

E nosso html de criação, mudamos a tipagem da data, para apenas “date”.

Em nosso componente.ts adicionamos a nossa variável date como uma string.

Para vincular a nossa variável “date”, precisamos converter para o formato “Date” conforme nosso Model do Ciclo de Pagamento, para conversão adicionamos uma nova data e passar a variável criada como string.

Antes de enviarmos nosso Ciclo de Pagamentos para o back-end, vamos testar nosso objeto, se ele está conforme o esperado. Para isso comentamos nosso serviço de envio e printamos com o console.log nosso objeto “cycle”.

Ao adicionarmos no Front o nosso atributo crédito e a data, ao clicar em Cadastrar ele já nos apresenta a estrutura desejada.

Após teste realizado, tiramos o comentário do service e vamos verificar se o nosso “create” está funcionando.

Incluímos em nossa aplicação uma data para o nosso Ciclo de pagamento e Nome e valor para nosso atributo do Crédito, após a inclusão do crédito, clicamos em Cadastrar para enviar para nosso banco de dados o nosso novo Ciclo cadastrado.

Aqui utilizaremos uma tabela do material para exibir os Créditos e Débitos do ciclo conforme formos adicionando na tela.

Primeiro importamos Material Table no app.module.ts.

Dentro do html do create-cycle, no mat-card de Créditos adicionamos a tag table.

Nessa tag:

- dataSource aponta para o array de credits no create-cycle.component.ts;

- matColumnDef define o nome da coluna;

- mat-cell vai é onde será especificado qual valor do array credits que será exibido;

- mat-header-row vai apontar para um array de Strings no create-cycle.component.ts. Nesse array estarão contidos os nomes das colunas que vai ser exibidas no header.

Feito isso a tabela estará funcionando na aplicação.
